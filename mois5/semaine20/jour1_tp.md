# TP Jour 1 : Le Grand Ménage

**Durée :** ~4h | **Fil Rouge :** Polissage et README final

---

## Exercice 1 : Chasse aux Warnings (1h30)

1. Active les options de compilation strictes dans `build.sbt` (`-Xfatal-warnings`).
2. Corrige chaque ligne de code qui génère un warning.
3. Vérifie que le projet compile parfaitement en "Strict Mode".

---

## Exercice 2 : Documentation d'Exécution (1h30)

1. Rédige un `README.md` exemplaire contenant :
   - Un schéma d'architecture.
   - Les pré-requis (Docker, Java).
   - Les commandes pour lancer la démo.
   - Comment accéder au Dashboard Grafana et au Swagger.

---

## Exercice 3 : Test de "Boîte Noire" (1h)

1. Supprime tout ton environnement Docker (`docker system prune -a`).
2. Suis exactement les instructions de ton propre README.
3. Si une seule étape échoue ou est floue, corrige ton README.

**Livrable** : Code source final (propre) et le fichier `README.md` complet.
