# TP Jour 4 : Jour de Gloire

**Durée :** Toute la journée

---

## Point de départ

- Utilise le **Kit 20.4** de `starter_kit_s20.md`.
- Vérifie le matériel, les accès et le plan de secours avant l'arrivée du jury.
- Désigne une personne chargée de noter les questions et les décisions.

## Exercice : La soutenance

1. Présente le besoin, les décisions, les preuves et les limites dans le temps prévu.
2. Exécute le scénario nominal.
3. Exécute une seule expérience de résilience préparée.
4. Réponds avec des faits ; indique clairement lorsqu'une réponse demande vérification.
5. Recueille les retours et les actions.

**Validation :** la grille contient temps, démonstrations réussies, questions ouvertes et actions assignées.

**Livrable** : support final, grille d'évaluation et compte rendu des retours.
